Methodology - One D Index

The One D Scorecard displays data indicators in a way that offers multiple angles of analysis by ranking metro regions, trending annual change, and indexing indicators within their priority areas across geographies. Data Driven Detroit (D3) developed the One D Index to summarize a single, comprehensive score that comprises the individual indicators that drive the priority areas.

The One D Index value is calculated by transforming the original values of individual indicators over a 0-5 range, with 0 as the lowest score and 5 as the highest. The comprehensive index value by priority area is then calculated by averaging the scores for the individual indicators within that priority area. Each indicator is equally weighted to comprise the priority area index, with the exception of three indicators broken out by race in Social Equity, where each of the three race categories comprise one third the weight of the total indicator. Similarly, the One D Index is calculated for each metro region by averaging the score for each priority area. Each priority area is equally weighted to comprise the One D Index score, independent of the number of indicators within that priority area.

The data available for download are the original values for each indicator as well as the One D Index and index score by priority area for each region.
